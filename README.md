# Data Dashboard Platform

This project is a data dashboard platform that allows users to explore datasets, generate visualizations, and analyze trends.

## Features

- **Data Querying**: Query datasets with filters and limits.
- **Visualizations**: Create and view charts/graphs.
- **Trend Analysis**: Perform trend analysis on datasets.
- **User Management**: Manage users and roles.
